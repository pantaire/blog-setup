# General
|   Function        |   Shortcut           |
|-------------------|---------------------|
|   Move window     |   right click drag  |
|   Open menu       |   right click       |
|   Select images   |   left click/drag   |
|   Focus images    |   double left click |
|   Zoom to pointer |   scroll wheel      |
|   Pan Scene       |   scroll click drag |
